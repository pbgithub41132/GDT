This Project was created by Paul Blumenthal using Unity Editor 2021.2.16f1.

Project/ contains everything required to import this project.
Screenshots/ contains screenshots of the built project running.
Shipping.zip contains a built version of the project including all required Unity runtime binaries.

Known issues:
    You may need to play (ctrl+p) or build and run (ctrl+b) the project once after importing the project and opening the 
    MainScene found in Project/Assets/Scenes/ in order for the buttons to be displayed properly within the Unity Editor.
    
So far the only implemented language localisation is german.
More languages may be added in the future.
